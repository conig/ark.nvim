# Inherited upstream reference

These files describe Positron, Jupyter, graphics-device, and variables-pane
surfaces inherited from `posit-dev/ark`. They are retained only for contributors
extracting reusable upstream code. They do not describe supported `ark.nvim`
product features.

Use `:help ark`, the repository `README.md`, and `docs/troubleshooting.md` for
the supported Neovim product.
